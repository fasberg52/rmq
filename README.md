# microservice-crypto

## Project setup for Order

```bash
$ pnpm install
```

## Compile and run the project

```bash
# development-SWC recomended

$ pnpm run swc

# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Run Migrations

```bash
# migration run

$ npm run migration:run

```
